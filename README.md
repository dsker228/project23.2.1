# Проект 23.2.1

# Конвертер валют с графическим интерфейсом

## 1. Постановка задачи

### Цель проекта
Разработать настольное приложение для конвертации валют с графическим интерфейсом, которое позволяет пользователям быстро и удобно переводить суммы из одной валюты в другую по актуальным курсам Центрального Банка РФ.
Задача выполнялась с написанием парсера данных с сайта myfin.by.

## 2. Инструкция по сборке и запуску

### Предварительные требования
1. Установленный Python версии 3.7 или выше
2. Установленный pip (менеджер пакетов Python)
3. Установленное приложение компилятор/интерпретатор (Pycharm, VS code)

### Шаг 1: Установка зависимостей

В командной строке вашего компьютера напишите по порядку:
1. ...python
2. ...pip install PyQt5
3. ...pip install bs4
4. ...pip install requests

### Шаг 2: Запуск кода

1. Откройте установленный компилятор/интерпретатор
2. Откройте файл с кодом конвертера
3. Запустите
